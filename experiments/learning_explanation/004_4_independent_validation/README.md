# 004-4: Independent Validation

**Status:** Not authorized

Independent validation begins only after a development method passes all frozen
gates and is repository-frozen. Before viewing the independent source, the
prompt, model, parameters, schemas, evaluator, rubric, and thresholds must be
fixed.

The independent benchmark must contain newly annotated Oracle Connections and
must not be used for prompt development. Passing this stage would validate only
limited Oracle-conditioned Learning Explanation, not Connection Discovery,
predicted-Connection explanation, or the end-to-end product pipeline.
